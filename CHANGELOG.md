# Changelog

## [1.0.0.0] - 2026-07-13

### Added

- Desktop client in C#/.NET 8 with Avalonia UI.
- FastAPI backend.
- PostgreSQL database.
- Redis for temporary state, rate limiting and anti-replay support.
- Client-side E2EE message flow.
- Device request signatures with Ed25519.
- WebSocket communication.
- Docker Compose infrastructure.
- Caddy reverse proxy.
- Cloudflare/TLS production setup.
- Public website with download area.
